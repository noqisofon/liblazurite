template<_Tree>
class lazurite::pattern::building::HierarchyBuilder< _Tree >

HierarchyPacker クラスは、ツリー構造の対象物を参考に、再帰を使って TreeView コントロールでのツリー構造を簡単に作成するための抽象クラスです。

此の例では、2 つ上のディレクトリをカレントとした木構造を作成します。 

 Exsample:
      HierarchyPacker<Tree> packer = new FileSystemPacker("../../");
      packer.each_pack();
      Tree tree = packer.result;
